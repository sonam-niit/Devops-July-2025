# EC2 monitoring using AWS CloudWatch

- Monitor EC2 Instance
- Monitor Network Traffic
- Create Cloudwatch Alarm
- Receive Email Notification

- Check Health Metrics
- Create Alarm incase instance become unhealthy
- Monitor Network Use (Network In/ Network out)
