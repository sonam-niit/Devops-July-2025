# practice question for Docker

1. Create one simple portfolio website using HTML/CSS

    - use some AI tool also for designing your website
    - once your website is ready
    - create Dockerfile
    - use nginx as  base image
    - copy your code to default location of nginx
    - then expose on port 80

    - build image
    - run image in once portfolio named container with external port 9090
    - check browser localhost:9090
    - you can access your potfolio here.
